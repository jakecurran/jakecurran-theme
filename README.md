# Usage

Requirements:

- Zola >= 0.7

First download this theme to your themes directory:

```bash
$ cd themes
$ git clone https://github.com/jakecurran/jakecurran-theme.git
```

and then enable it in your `config.toml`:

```toml
theme = "jakecurran"
```
